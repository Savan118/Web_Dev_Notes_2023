CSS Layouts

  • The CSS float properties specifies how an element should float.
  • The CSS clear properties specifies what elements can float besides the cleared element and on which side.

Float properties

  • The float properties is used for positioning and formatting the content.
  • For example: lets take and image and apply the float properties on it.
  • float properties have following values.
    1. left: the element float to the left of its container.
    2. right: the element float to the right of its container.
    3. none: the element does not float (this is default property).
    4. inherit: the element inherit the float value of its parent.

Examples of float properties

1. float: left;

<img src="URL of image" style="height: 150px; width: 150px; float: left; padding: 2px; margin: 2px;">
